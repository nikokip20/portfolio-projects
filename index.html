Analysis Data

As a recent graduate eager to embark on a career as a data analyst, I am excited to dive into the world of data analysis and visualization. For this project, I will be analyzing a dataset that contains information about Netflix shows released between 2008 and 2021. This dataset offers a fascinating glimpse into how Netflix has evolved its content strategy and the variety of video-on-demand options it has provided to viewers over the year.The dataset includes detailed information about individual movies and series, such as the names of actors and directors, release dates, and various categorical variables like content rating (e.g., TV-Y, PG) and type (TV Show or Movie). With over 7000 entries, it offers a substantial amount of data to work with. However, there are some challenges, such as missing information for directors and cast members, which have been addressed by replacing missing values with "unknown" or by excluding overly detailed descriptions to streamline the analysis.Through data visualization, I aim to uncover meaningful patterns and trends in the dataset, starting with an analysis of the number of shows released each year. This project not only serves as a valuable learning experience but also demonstrates my analytical skills and passion for turning data into actionable insights.

Dissertation

Background and Motivation :
The rapid evolution of information technologies has integrated computers deeply into various facets of our lives, from personal use to professional applications. With this increased reliance on digital systems, the prevalence of high-technology crimes has also surged, necessitating advanced methods for data recovery and forensic analysis. Tapping and analyzing data in storage devices has become crucial for uncovering evidence of criminal activities, making file system analysis an indispensable skill for cybersecurity professionals.
Objective:
This dissertation focuses on the analysis of NTFS file systems and the development of software tools using Windows PowerShell to recover or find information within file systems. Specifically, the project aims to recover deleted files, locate specific files—including those hidden using alternative data streams (ADS)—and identify potentially harmful files within a system. The objective is to provide a robust solution using native Windows tools without relying on third-party software, enhancing accessibility and ease of use for forensic investigators and IT professionals.
